# Playwright Practice Automation

Projeto de estudos utilizando Playwright para automação de testes web.

## Tecnologias

- Playwright
- JavaScript
- Node.js

## Cenários Automatizados

### Login
- Login válido
- Login inválido (senha incorreta)
- Login inválido (usuário incorreto)

## Como executar

npm install
npx playwright test

## Relatórios

npx playwright show-report
